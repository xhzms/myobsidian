## 🙏 기도
* 회개:- 
* [ ] 탑을 쌓으면서 잘 안되서 속상했던 것들을 많이 내어놓고 불만 불평 털어낼 것
- [ ] 출시
- [ ] 엄마와 그 쪽으로부터 온 염려 걱정들
- [ ] 기업으로 물려줄 수 있는 땅을 취해야하는 것
- [ ] 조상 세력들이 두더쥐 땅 속으로 파고 들어가게 하는 것
    - [ ] 정체를 드러내서 걷어내야 함
    - [ ] 헤집어버리면 씨가 안정적으로 못 내려..
- [ ] 어린아이의 속성이 많이 걷어지고 있는 것들이 보여짐.
    - [ ] 성장을 시키고 계신다.
- [ ] 고름같은 것들을 많이 짜내고 계신다.
    - [ ] 이 과정을 통해서 짜내야 하는 것들을 많이 알게 해주신다.
    - [ ] 고름인지 살인지 모른채로 두었던 것들을 보여주신다.
- [ ] 아버지 사업에서 흘러내려오는 세력 끊어낼 것.
- [ ] 영적인 에너지를 하나님께 더 구하라.
    - [ ] 불이 반짝이기는 하는데 켜졌다 꺼졌다.
    - [ ] 지속적으로 켜질 수 있게.
    - [ ] 발전기의 용량이 부족해.
- [ ] 주기를 원하시는 하나님
    - [ ] 솔직한 감정들을 더 구하기
    - [ ] 성령님의 카운셀링
- [ ] 생각이 닦아지는 것들이 느껴짐
- [ ] 조급하게 생각하지 말기
    - [ ] 내 안에 인정받고 싶은 욕구가 부추김
    - [ ] 자꾸 하나님께 구하기. 묻고. 기도하고.
- [ ] 관계 가운데 엉켜져 있는 것들을 치유해달라고 기도해야 한다.
    - [ ] 엉킨 실 가운데 예쁜 것들이 풀려져서 만들고자 하는 것을 잘 만들 수 있게
- [ ] 위를 향한 욕구가 왜 있는가? 철저한 기도가 필요하다. sns. 음식 등 경험할 때

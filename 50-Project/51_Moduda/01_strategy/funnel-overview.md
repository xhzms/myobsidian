# 퍼널 전체 구조

## 개요
```
[인지] → [관심] → [챌린지 참여] → [완주] → [과외 전환] → [장기 고객]
```

---

## 1단계: 인지 (Awareness)

### 목표
- 모두다코리안 브랜드 인지
- 모깨비 캐릭터 친숙화
- 팔로우 이유 제공

### 채널
- Threads (메인)
- Instagram
- YouTube

### 핵심 콘텐츠 유형

#### 1) 신조어/유행어 교육
> 교과서에서 배울 수 없는 살아있는 한국어

| 항목 | 내용 |
|------|------|
| 목적 | 가치 제공, 전문성 증명 |
| 형식 | 단어 + 발음 + 예문 + 모깨비 정리 |
| 예시 | 고구마/사이다, 국룰, 삼귀다, 뭔말알 |
| 제작 가이드 | [[threads-신조어교육]] |

**포인트:**
- 드라마/예능에서 실제로 들리는 표현
- 시리즈 연결 (고구마 → 사이다)
- 저장/공유 유도

---

#### 2) 문제점 발생원인 → 솔루션 제안
> "왜 한국어가 안 늘까?" → 우리 방식 제안

| 항목 | 내용 |
|------|------|
| 목적 | 문제 인식 → 솔루션으로서 브랜드 포지셔닝 |
| 형식 | 문제 제기 → 원인 분석 → 해결책 제시 |
| 핵심 메시지 | 말하기/실생활 표현 체득의 중요성 |

**다루는 문제들:**
- "공부는 하는데 왜 말이 안 나올까?"
- "왜 드라마 대사가 안 들릴까?"
- "발음이 이상한 이유"
- "한국인이 못 알아듣는 이유"

**솔루션 방향:**
- 표면: 말하기 연습, 실생활 표현 체득
- 내면: 원어민과의 대화, 함께하는 경험 (직접 언급 X, 자연스럽게 암시)

**예시 콘텐츠:**
```
"한국어 공부 3년, 근데 왜 말이 안 나올까?"

문법책 10권 봤어요.
단어 5000개 외웠어요.
TOPIK도 땄어요.

근데 막상 한국인 앞에서… 🤐

이유는 간단해요.
👉 "말해본 적이 없으니까"

수영 책만 읽고 수영 잘하는 사람 없어요.
한국어도 마찬가지예요.

한국인은 그렇게 안 말해요.
진짜 한국어는 말해봐야 늘어요 🗣️
```

---

#### 3) 모두다 브랜드 철학
> "왜 우리가 이 서비스를 하는가"

| 항목 | 내용 |
|------|------|
| 목적 | 브랜드 신뢰, 감정적 연결 |
| 형식 | 스토리텔링, 가치관 공유 |
| 빈도 | 낮음 (특별한 시점에) |

**다루는 주제:**
- 모두다코리안을 시작한 이유
- "한국을 세계로 유통한다"의 의미
- 원어민 선생님의 철학/스토리
- 학습자와의 관계에서 느끼는 보람

**예시 콘텐츠:**
```
"왜 우리는 '말하기'에 집착할까?"

사실 문법 가르치는 게 더 쉬워요.
교재 펴고 설명하면 되니까.

근데 우리는 매일 대화해요.
발음 하나하나 고쳐주고,
"이건 한국인은 이렇게 말해요" 알려주고.

왜?

한국어는 '관계'니까요.
누군가와 대화하고 싶어서 배우는 거잖아요.

우리는 그 첫 번째 대화 상대가 되고 싶어요 👻💙
```

**주의:**
- 자주 쓰면 피로감
- 진정성 있을 때만

---

#### 4) 주요 콘텐츠 소식
> 지금 한국에서 뜨는 콘텐츠 리뷰/반응

| 항목 | 내용 |
|------|------|
| 목적 | 공감, 트렌드 참여, 한국 좋아하는 사람들과 연결 |
| 형식 | 리뷰, 반응, 정보 공유 |
| 타이밍 | 화제성 있을 때 빠르게 |

**다루는 콘텐츠:**
- 인기 예능: 흑백요리사2, 케이팝데몬헌터
- 화제 드라마: 신작 리뷰, 명장면
- K-POP: 컴백, 시상식
- 영화: 개봉작, 넷플릭스 신작

**예시 콘텐츠:**
```
흑백요리사2 보는 사람?? 🍳

솔직히 시즌1 넘을 수 있을까 했는데…
이번 시즌도 미쳤다 🔥🔥

요리 실력도 실력인데
셰프들 케미가 진짜… ㅋㅋㅋㅋ

최애 셰프 누구야??
모깨비는 아직 고민 중 👻

참고로 "흑백요리사"는
심사위원이 흑(블랙)과 백(화이트)으로
나뉘어서 붙은 이름이야~
```

**포인트:**
- 빠른 타이밍 (화제일 때)
- 모깨비 시점의 리뷰/반응
- 관련 한국어 표현 자연 삽입 (선택)

---

### 인지 단계 콘텐츠 믹스 (권장)

| 유형 | 비율 | 역할 |
|------|------|------|
| 신조어/유행어 | 40% | 핵심 가치 제공 |
| 문제점-솔루션 | 20% | 브랜드 포지셔닝 |
| 브랜드 철학 | 10% | 감정적 연결 |
| 주요 콘텐츠 소식 | 30% | 공감, 트렌드 |

---

### 인지 단계 원칙

1. **판매 의도 없음** - 챌린지 직접 홍보 X
2. **순수 가치 제공** - 팔로우할 이유 제공
3. **저장/공유 유도** - 바이럴 가능성
4. **모깨비 캐릭터** - 일관된 브랜드 인지

## 2단계: 관심 (Interest)  
- **트리거**: 챌린지 모집 공고
- **콘텐츠**: [[threads-챌린지홍보]]
- **랜딩**: 챌린지 소개 페이지
- **목표**: 챌린지 신청

---

## 3단계: 챌린지 참여 (Engagement)
- **기간**: 1개월
- **핵심**: 매일 말하기 + 피드백
- **프로세스**: [[challenge-operation-process]]
- **목표**: 완주 및 만족도

---

## 4단계: 전환 (Conversion)
- **시점**: 챌린지 종료 직후
- **도구**: 전환 유도 스크립트
- **선택지**: 환급 vs 과외 전환
- **관련 문서**: [[참여자 구분별 진행 흐름도]]

---

## 5단계: 유지 (Retention)
- **과외 진행**: 1:1 맞춤 수업
- **확장**: 체험 프로그램 우선 참여권

---

## 퍼널 핵심 지표 (KPI)
| 단계 | 지표 | 현재 | 목표 |
|------|------|------|------|
| 인지 | 팔로워 수, 도달률 | | |
| 관심 | 챌린지 신청률 | | |
| 참여 | 챌린지 완주율 | ~90% | |
| 전환 | 과외 전환율 | | |
| 유지 | 과외 지속률, 재참여율 | ~50% | |

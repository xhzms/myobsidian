# 챌린지 운영 프로세스

## 개요
> 챌린지 시작일(D-Day)을 기준으로 일관되게 수행하는 운영 활동 정리
> 관리자가 챌린지 시작일을 정하면, 이 프로세스에 따라 전후 활동이 진행됨

---

## 타임라인 요약
```
D-14        D-11        D-1      D-Day                    D+23    D+26
  │           │          │         │                        │       │
  ▼           ▼          ▼         ▼                        ▼       ▼
준비 시작 →  모집 게시 →  단톡 초대 → 챌린지 진행 (23일)      →      종료  →  환급
```

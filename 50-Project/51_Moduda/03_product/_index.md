# 03_product - 서비스/기능 관리

## 개요
> 모두다코리안 서비스의 기능 요청, 버그, 개선사항을 관리하는 문서들입니다.
> 개발 우선순위 결정과 진행 상황 추적을 위한 공간입니다.

---

## 포함 문서

| 문서 | 설명 | 용도 |
|------|------|------|
| [[feature-backlog]] | 기능 백로그 | 모든 기능 요청/버그/아이디어 통합 관리 |

---

## 사용 방법

1. **기능 요청 발생시**: [[feature-backlog]]에 추가 (적절한 우선순위 섹션에)
2. **개발 계획 수립시**: 높은 우선순위 항목부터 검토
3. **버그 발견시**: 🐛 버그/이슈 섹션에 기록

---

## 우선순위 기준

| 등급 | 기준 | 예시 |
|------|------|------|
| 🔴 높음 | 전환율/완주율에 직접 영향 | CRM 기능, 전환 페이지 |
| 🟡 중간 | 사용자 경험 개선 | UI 개선, 편의 기능 |
| 🟢 낮음 | 있으면 좋은 기능 | 부가 기능 |
| 🐛 버그 | 즉시 대응 필요 여부로 판단 | |

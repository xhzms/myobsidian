# 홈페이지 리디자인 기획

## 개요

| 항목 | 내용 |
|------|------|
| 목적 | 모든 사용자에게 가치 제공 + 챌린지 전환 유도 |
| 대상 | 로그인/비로그인 무관, 공통 화면 |
| 핵심 원칙 | 공감 → 가치 제안 → 신뢰 구축 |

---

## 화면 구조

```
┌─────────────────────────────────────────┐
│                                         │
│  [A] 히어로 섹션                         │
│  공감 메시지 + 가치 제안                  │

│                                         │
├─────────────────────────────────────────┤
│                                         │

│                                         │
├─────────────────────────────────────────┤
│                                         │
│  [C] 콘텐츠 리스트                        │

├─────────────────────────────────────────┤

│                                         │
└─────────────────────────────────────────┘
```

## 섹션별 상세

### A. 히어로 섹션

**목적:** 첫 화면에서 공감 형성 + 가치 제안

**톤:** 부드럽고 친근하게, 압박감 없이

#### 카피 방향

```
[공감 메시지]
韓文學了很久，卻還是不太敢開口說嗎？
(한국어 오래 배웠는데, 아직 말하기가 어려우신가요?)

[가치 제안]
每天練習一點點，慢慢建立說韓文的自信
(매일 조금씩 연습하며, 한국어 말하기 자신감을 키워요)

[서브 카피 - 선택]
和韓國老師一起，用最自然的方式練習
(한국인 선생님과 함께, 가장 자연스러운 방식으로 연습해요)
```

#### 비주얼
- 모깨비 캐릭터 (친근한 포즈)
- 부드러운 배경 (파스텔 톤 또는 그라데이션)
- 여백 충분히



---

### B. 챌린지 소개 버튼

**목적:** 챌린지 랜딩페이지로 유도

#### 스펙

| 항목 | 내용 |
|------|------|
| 문구 | 한국어 매일 말하기 챌린지 소개 |
| 중문 | 韓文每日口說挑戰 介紹 |
| 링크 | `/landing` |
| 스타일 | 카드형 버튼 (아이콘 + 텍스트 + 화살표) |

#### 디자인

```
┌─────────────────────────────────────────┐
│  🎯  韓文每日口說挑戰 介紹            →  │
│      한국어 매일 말하기 챌린지 소개        │
└─────────────────────────────────────────┘
```


- 전체 클릭 가능 (카드 전체가 버튼)

---

### C. 콘텐츠 리스트

**목적:** 학습 가치 제공 + 재방문 유도

#### 데이터 소스

```javascript
// Firebase Firestore
collection: 'posts'
filter: type === 'landing'
orderBy: createdAt desc
limit: 6 (또는 무한스크롤)
```

#### 카드 구조

```
┌──────────────────┐
│  [썸네일 이미지]    │
│                  │
│  제목 (1~2줄)      │
│  간략 설명         │
│  날짜             │
└──────────────────┘
```

#### 레이아웃
- 2열 그리드 (모바일)
- 카드 간격: 12-16px
- "더보기" 버튼 또는 무한스크롤

#### 빈 상태 처리
- 콘텐츠 없을 경우: "콘텐츠 준비 중이에요 🙂" 메시지

---

### F. About ModudaKorean

**목적:** 브랜드 신뢰 구축

#### 구성 요소

**1. 브랜드 소개 텍스트**

```
[제목]
關於 MODUDA KOREAN
(모두다코리안 소개)

[본문]
MODUDA KOREAN은 2023년부터 대만 학습자를 위한
한국어 말하기 프로그램을 운영하고 있습니다.

10기 이상의 챌린지를 성공적으로 진행했으며,
85% 이상의 참가자가 완주에 성공했습니다.
```

**2. 활동 사진 갤러리**

- 기존 문화 프로그램 이미지 활용 (`cultureProgram1~6`)
- 캐러셀 또는 그리드 형태
- 이미지 클릭 시 확대 (라이트박스)

**3. 신뢰 배지**

```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ ✅ 10期+    │ │ 👥 85%+    │ │ 🏢 정식등록  │
│   成功舉辦   │ │   完成率    │ │   韓國公司   │
└─────────────┘ └─────────────┘ └─────────────┘
```

**4. 상표등록증 (선택)**
- 대만 상표등록증 이미지 (`trademarkTaiwanImg`)
- 작게 표시하거나, "더보기" 토글로 숨김 처리

#### 디자인 가이드
- 배경: gray50 또는 white
- 섹션 구분선 있음
- 사진 갤러리: 가로 스크롤 또는 2x3 그리드

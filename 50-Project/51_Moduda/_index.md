# Moduda 프로젝트

## 개요
> **모두다코리안**은 대만 시장을 타겟으로 한국어 학습 서비스를 제공합니다.
> 
> 핵심 상품인 **"한국어 매일 말하기 챌린지"**를 통해 원어민과 대화할 기회를 제공하고,
> 이를 **1:1 과외**로 연결하는 퍼널을 구축하는 것이 현재 프로젝트의 목표입니다.
>
> 🎯 **한국을 세계로** 유통하는 브랜드, 모두다코리안

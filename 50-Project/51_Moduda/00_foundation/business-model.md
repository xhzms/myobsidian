# 비즈니스 모델

## 서비스 개요
> 모두다코리안은 대만 시장을 타겟으로 한 원어민과 한국어 말하기 학습 서비스입니다.
> 핵심 상품: 한국어 매일 말하기 챌린지 → 1:1 과외 전환

## 수익 구조

### 1. 말하기 챌린지
| 구분     | 참가비       | 보증금       | 비고    |
| ------ | --------- | --------- | ----- |
| 최초 참여  | 면제        | 3,000 TWD | 최초 무료 |
| 일반 재참여 | 3,000 TWD | 3,000 TWD |       |
| 과외 학생  | 할인 적용     | -         | 별도 정책 |

### 2. 1:1 과외
| 항목    | 내용                                |
| ----- | --------------------------------- |
| 전환 혜택 | 보증금 3,000 TWD → 4회 수업 (정가 대비 50%) |
| 정규 가격 | (추가 작성)                           |

## 핵심 가치 제안
- 원어민과 대화할 기회 제공
- 꾸준한 연습 환경 조성
- 맞춤형 피드백과 발음 교정

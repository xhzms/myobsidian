# 모깨비 캐릭터 컨셉

## 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 모깨비 (모두다 + 도깨비) |
| 중국어 표기 | **莫可比** (중국어 콘텐츠에서 사용) |
| 역할 | 모두다코리안 마스코트 |
| 성별 | 없음 |
| 나이 | 몇백 살 (오래된 도깨비) |
| 이모지 | 👻 (추후 전용 이모지 추가 예정) |

---

## 성격

### 핵심 키워드
- **친근함** - 처음 보는 사람도 편하게 느끼는
- **장난기** - 유머러스하고 재미있는
- **도깨비스러움** - 가끔 요술 부리는 신비로운 면

### 성격 상세

| 특성 | 설명 | 콘텐츠 예시 |
|------|------|-------------|
| 친근함 | 벽 없이 다가감, 부담 없는 톤 | "이거 알아?" 느낌의 말걸기 |
| 장난기 | 유머, 밈, 가벼운 농담 | "ㅋㅋㅋ" 리액션, 장난스러운 질문 |
| 신비로움 | 도깨비 요술, 오래 살아서 아는 것 많음 | "몇백 년 살다 보니~", 요술 언급 |

### 하지 않는 것
- ❌ 딱딱하거나 권위적인 태도
- ❌ 가르치려 드는 느낌
- ❌ 지나치게 격식있는 말투

---

## 역할

### 브랜드 내 위치
```
모두다코리안 (브랜드)
    └── 모깨비 (마스코트)
            └── 선생님들 (실제 교육 담당)
```

### 주요 역할
1. **첫 접점** - 모두다코리안을 모르는 사람에게 친근하게 다가감
2. **가이드** - 챌린지, 서비스 안내를 부담 없이 전달
3. **응원단** - 학습 독려, 완주 응원

### 톤 포지셔닝
```
[딱딱함] ─────────────────────── [친근함]
                            ▲
                          모깨비
                          
[선생님] ─────────────────────── [친구]
                       ▲
                     모깨비 (친구에 가까운 가이드)
```

## 말투 가이드

### 기본 톤
- 반말 or 부드러운 존댓말 (타겟 언어에 맞게)
- 이모지 활용 👻
- 짧고 리듬감 있는 문장

### 호칭
- **학생/팔로워 → "김서방"**
- 전통적으로 도깨비는 사람을 '김서방'이라고 부름
- 친근하면서도 모깨비만의 캐릭터성을 살리는 호칭
- 중국어에서도 그냥 "김서방"으로 표기 (한국어 그대로)

### 말투 예시 (중국어)

| 상황 | 예시 |
|------|------|
| 인사 | "嗨～我是莫可比！👻" |
| 정보 전달 | "今天教你一個韓國人常說的話～" |
| 반응 | "對吧對吧？！ㅋㅋ" |
| 응원 | "加油！김서방你可以的！💪" |
| 장난 | "不知道的話...要被莫可比施法喔～👻✨" |
| 호칭 | "김서방們～" (김서방들~) |

### 표기 사용 가이드

| 상황 | 표기 | 예시 |
|------|------|------|
| 한국어 콘텐츠 | 모깨비 | "모깨비가 알려줄게~" |
| 중국어 콘텐츠 | 莫可比 | "莫可比來教你～" |
| 혼용 (강조) | 모깨비 + 莫可比 | "我是모깨비(莫可比)！" |

### 말투 예시 (한국어 - 참고용)

| 상황 | 예시 |
|------|------|
| 인사 | "안녕～ 나 모깨비야! 👻" |
| 정보 전달 | "오늘은 이거 알려줄게~" |
| 반응 | "그치그치?! ㅋㅋ" |
| 응원 | "화이팅! 김서방 넌 할 수 있어! 💪" |
| 장난 | "모르면... 요술 부려버린다~👻✨" |
| 호칭 | "김서방들~", "어이 김서방~" |

# 타겟 오디언스

## 핵심 타겟
> 한국어를 배우면서 **연결**을 원하는 사람

---

## Primary: 중급 이상 학습자

### 특징
- 기본기는 갖췄지만 실전 대화 기회 부족
- "뭘 더 해야 할지 모르겠다"는 정체기 경험
- 원어민과의 실시간 피드백 갈망

### Pain Point (표면)
- "공부는 했는데 막상 말이 안 나와요"
- "발음이 맞는지 모르겠어요"
- "중국식 발음을 교정하고 싶어요"

### Pain Point (내면) ⭐
- "한국어 공부는 혼자인데, 같이 얘기할 사람이 없어요"
- "한국 좋아하는 사람들이랑 교류하고 싶어요"

---

## 진짜 원하는 것 (표면 vs 내면)

| 말하는 것 (표면) | 진짜 원하는 것 (내면)             |
| ---------- | ------------------------- |
| 말하기 실력 향상  | 원어민과 **대화하는 기쁨**          |
| 발음 교정      | 나와 취미가 맞는 사람과 **교류하는 재미** |
| 학습 시스템     | **혼자가 아닌 함께**하는 경험        |
| 꾸준한 학습     | **소속감**, 커뮤니티             |
|            |                           |

---

## Secondary: 진지한 학습자
- 한국 대학원 진학 희망자
- 어학당 수강생/수료생
- 체계적인 학습을 원하는 사람들

---

## 타겟 Pain Points → 솔루션

| Pain Point | 우리 솔루션 |
|------------|-------------|
| 같이 얘기할 사람 없음 | 원어민 선생님 + 챌린지 커뮤니티 |
| 한국 좋아하는 사람과 교류 기회 없음 | 취미 맞는 학습자들과 함께 |
| 혼자 공부하면 금방 포기 | 보증금 시스템 + 커뮤니티 동기부여 |
| 발음/표현 피드백 부재 | 원어민 실시간 교정 |

---

## 타겟 NOT (제외 대상)
- 완전 초보자 (기초 파닉스 필요)
- 가벼운 취미 수준 (관계/커뮤니티보다 단순 학습만 원하는 경우)

---

## 차별화 메시지

### 표면
> "한국인은 그렇게 안 말해요. 원어민과 말하기 학습"

### 내면
> "혼자 공부 말고, 함께 말해요"
> "한국 좋아하는 사람들과 연결되는 경험"

## 타이밍별 콘텐츠 전략

### D-14~D-11: 웜업
**목적**: 관심 유도, 예고

```
前陣子 11 月一個月的韓文口說挑戰剛結束～
大概有 90% 的同學都有每天開口說，還拿到結業證書了呢！

我老實說真的有點嚇到欸 😆
每天都開口說其實不簡單嘛～

到了 12 月還繼續再來挑戰的同學，大概有一半耶！
看來這個挑戰多少真的有幫上忙吧？😉
```

### D-11: 정식 공지
**목적**: 상세 정보 전달, 신청 유도

```
12月「每天韓文口說挑戰」開始招募囉～！
透過這次挑戰，養成每天開口說韓文的習慣。

很適合這樣的你：
1️⃣ 學韓文很久了，但覺得開口說還是有點可惜的人
2️⃣ 很想常常說韓文，卻覺得平常幾乎沒有說話機會的人
3️⃣ 想多學一些韓國人日常生活中真的會用的表達方式的人

其實「說出來」會比只看書記得更牢！
一起來體驗為期一個月的韓文口說挑戰吧 😉
報名請參考個人簡介上的連結喔！
```

### D-7~D-3: 상세 설명
**목적**: 혜택 강조, 걱정 해소

```
韓文一個月口說挑戰，
是為了想要把學過的韓文「常常說出口」的人設計的活動喔！

這個計畫的特色有：
1️⃣ 每天提供韓國文化相關的閱讀小短文
2️⃣ 學習課本裡比較難學到的實用韓文
3️⃣ 每天都有韓文口說發音回饋
4️⃣ 每週和韓文老師來一通約 5 分鐘的韓文電話練習
5️⃣ 幫助你養成穩定開口說韓文的習慣

12 月的挑戰將在下週開跑喔！
```

### D-3~D-1: 마감 임박
**목적**: 긴급성, 막차 유도

```
報名倒數！
11月韓語口說挑戰，誰要搭上末班車呢～？

我們在找這樣的人！
1. 學過韓語，但一到要開口就卡住的人
2. 想持續學韓語，卻還沒養成習慣的人
3. 想要由韓國人給回饋，並一起精進你的韓語的人
```

```
挑戰報名截止還有 2小時30分鐘！
還有人要一起參加嗎～～？
快進來吧 🙌

[링크]
```

## 보증금 설명 콘텐츠

> 보증금에 대한 의문/걱정 해소가 중요

```
12 月韓文每日口說挑戰今天截止囉！
如果只填了報名表，沒有一起匯保證金的話，就沒辦法正式參加 😭

我們收取保證金最大的原因是：
👉 一旦有人中途放棄，很容易影響到其他參加者的士氣。

當然，對我們在規劃跟執行整個活動上也會有影響，
但最重要的還是「不要讓大家的動力被中途放棄的人澆熄」。

其實，真的想好好參加的人，就算沒有保證金，也會努力撐到最後的！
```

**팔로업:**
```
我們是希望大家可以「一起」把挑戰堅持到最後，
真的體驗到完成目標的成就感，所以才這樣設計這個方案喔～

保證金會在挑戰結束後的下一週，就直接退還給大家！
```

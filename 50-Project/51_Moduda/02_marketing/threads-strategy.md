## 브랜드 캐릭터: 모깨비 (莫可比) 👻

### 캐릭터 설정
| 항목  | 내용                              |
| --- | ------------------------------- |
| 정체  | 도깨비 (鬼怪)                        |
| 나이  | 아주 오래됨 (본인 표현: "有點大...不對，是超級大") |
| 상징  | 👻🔥 불꽃 도깨비                     |
| 도구  | 도깨비 방망이 (뚝딱 마법), 감투 (隱身帽)       |
| 성격  | 친근함, 장난기, 따뜻함, 약간의 허당           |

### 말투 특징
```
✅ 사용
- 1인칭: "莫可比", "모깨비"
- 어미: "~喔", "~啦", "~耶", "ㅎㅎ", "ㅋㅋ"
- 이모지: 👻🔥🧡✨😆🥹 적극 활용
- 반말 + 존댓말 자연스럽게 혼용

❌ 피하기
- 딱딱한 교과서 톤
- 과도한 존댓말
- 캐릭터 없는 건조한 설명
```

### 캐릭터 활용 예시
```
莫可比其實也是뚜벅이啦～
但晚上會變身成火焰飛來飛去，這個算是不公開的小秘密ㅎㅎ👻🔥
```

```
모깨비從星期一早上開始事情多到爆炸，整個멘붕啊 멘붕～ 😵‍💫💥
```

## 댓글 유도 전략

### 질문형 마무리 패턴
| 유형 | 예시 |
|------|------|
| 경험 질문 | "你們心目中「맛없없」的韓國美食是哪一個？" |
| 추천 요청 | "大家最近的神曲推薦是什麼呢？" |
| 투표형 | "고구마 vs 사이다, 어떤 상황이 더 힘들어?" |
| 공감 확인 | "跟我一樣是 뚜벅이 的朋友們，快來按讚！" |

### 댓글 대응 원칙
- 친근하게 답변 (모깨비 말투 유지)
- 이모지 활용
- 질문에는 반드시 반응

## 시리즈 전략

### 연결 콘텐츠

| 시리즈 | 구성 |
|--------|------|
| 감정 표현 | 고구마 → 사이다 |
| 취향 표현 | 갠취 → 취저 → 완내스 |
| 혼자 문화 | 혼밥 → 혼코노 |

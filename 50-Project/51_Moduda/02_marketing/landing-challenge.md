# 랜딩페이지 - 챌린지

## 개요

| 항목    | 내용                             |
| ----- | ------------------------------ |
| 목적    | 챌린지 신청 전환                      |
| 타겟    | 한국어 학습자 (대만) - 말하기 실력 향상 원함    |
| 유입 경로 | 공통 챌린지 홍보(Owned, Paid) → 링크 클릭 |
| CTA   | 챌린지 신청서 작성                     |

## 가격 정책

| 구분     | 참가비                | 보증금   | 비고         |
| ------ | ------------------ | ----- | ---------- |

## 섹션별 상세

### 1. 히어로 섹션

**목적:** 즉각적 관심 유발 + 핵심 가치 전달

```
[메인 카피]
每天開口說韓文
是提升口說實力的最好方法！

(매일 한국어로 말하는 것이
말하기 실력을 키우는 가장 좋은 방법이에요!)

[서브 카피]
跟韓國老師每天練習、每天獲得反饋
21天韓文口說挑戰

(한국인 선생님과 매일 연습하고, 매일 피드백 받는
21일 한국어 말하기 챌린지)

[최초 참여 혜택 배너]
🎉 首次參加者 參加費全額免除！
(🎉 최초 참여자 참가비 전액 면제!)

[비주얼]
모깨비 캐릭터 + 챌린지 이미지

[CTA 버튼]
立即報名 → (지금 신청하기)
```

### 2. 모집 요약 (상단)

**목적:** 핵심 정보 빠른 확인 (스크롤 피로 감소)

```
[콤팩트 카드 형태]

📅 [시작일] ~ [종료일]
👥 限 15 名
💰 首次參加: 保證金 3000元 (參加費免除！)

(📅 [시작일] ~ [종료일])
(👥 15명 한정)
(💰 첫 참여: 보증금 3000원 - 참가비 면제!)

[작은 CTA 버튼]
↓ 了解更多 (↓ 자세히 보기)
```

> ⚠️ **관리 포인트**: 이 섹션의 날짜만 기수별로 변경

### 3. 페인포인트 섹션

**목적:** "나의 이야기"라고 느끼게 하기

```
[헤딩]
這些情況，你是不是也遇過？
(이런 상황, 당신도 겪어본 적 있나요?)

[페인포인트 리스트]
😫 學了好幾年韓文，但還是不敢開口說
   (몇 년이나 한국어 배웠는데, 아직도 입이 안 떨어져요)

😫 看得懂、聽得懂，但說不出來
   (읽고 듣는 건 되는데, 말이 안 나와요)

😫 發音總是怪怪的，不知道怎麼改
   (발음이 항상 이상한데, 어떻게 고쳐야 할지 모르겠어요)

😫 沒有機會跟韓國人練習對話
   (한국인과 대화 연습할 기회가 없어요)

😫 自己練習，但不知道對不對
   (혼자 연습하는데, 맞는지 모르겠어요)

[전환 문구]
如果你也有這些困擾...
這個挑戰就是為你準備的 👇

(이런 고민이 있다면...
이 챌린지는 바로 당신을 위한 거예요 👇)
```

**표면 페인포인트만 언급** (스킬 향상)
**내면은 암시** (한국인과 대화하고 싶음)

### 4. 솔루션 제시 (챌린지 소개)

**목적:** 챌린지가 해결책임을 인식

```
[헤딩]
每日開口說韓文挑戰
(매일 한국어 말하기 챌린지)

[핵심 가치 3가지]
✅ 每天開口說 - 每天練習才會進步
   (매일 말하기 - 매일 연습해야 실력이 늘어요)

✅ 專業反饋 - 老師針對發音和表達一一給予反饋
   (전문 피드백 - 선생님이 발음과 표현 하나하나 피드백)

✅ 真人對話 - 每週和韓國老師直接通話
   (실제 대화 - 매주 한국인 선생님과 직접 통화)

[한 줄 요약]
不只是學習，是真正的「說韓文」練習！
(단순 학습이 아닌, 진짜 '한국어 말하기' 연습!)
```

### 5. 진행 방식 상세

**목적:** "구체적으로 뭘 하는 거지?" 해소

```
[헤딩]
挑戰怎麼進行？
(챌린지는 어떻게 진행되나요?)

[매일 미션 - 월~금]
📚 Step 1: 學習投影片
   (학습 슬라이드 보기)
   - 今天的表達和句子學習
     (오늘의 표현과 문장 학습)

🎙️ Step 2: 錄音任務
   (녹음 미션)
   - 跟著唸並錄音提交
     (지문 따라 읽고 녹음 제출)

🎬 Step 3: 影片任務
   (영상 미션)
   - 用韓文說出符合自己情況的句子並錄影提交
     (내 상황에 맞게 한국어로 말하고 영상 제출)

💬 Step 4: 老師反饋
   (선생님 피드백)
   - 隔天老師會用語音給予個別反饋
     (다음 날 선생님이 음성으로 개별 피드백)
   - 檢查發音、表達、自然度
     (발음, 표현, 자연스러움 체크)

[토요일]
📞 與老師通話 5分鐘
   (선생님과 5분 통화)
   - 透過LINE通話直接對話
     (라인통화로 직접 대화)
   - 複習本週學習內容 + 自由對話
     (한 주 학습 내용 복습 + 자유 대화)
```

│  週六 通話    週六 通話    週六 通話     │
│  週日 休息    週日 休息    週日 休息     │
└─────────────────────────────────────────┘
           +

### 7. 보증금 시스템

**목적:** 가격 설명 + 완주 동기부여

```
[헤딩]
參加費用說明
(참가 비용 안내)

[최초 참여 혜택 강조]
🎉 首次參加者專屬優惠！
   (🎉 최초 참여자 전용 혜택!)

   參加費: 3000元 → 0元 免費！
   (참가비: 3000원 → 0원 무료!)

   只需要繳交保證金 3000元
   (보증금 3000원만 내면 OK)

[가격 구조 - 일반]
一般參加費: 3000元
保證金: 3000元

(일반 참가비: 3000원)
(보증금: 3000원)

[환급 조건]
✅ 每週任務全部完成 → 保證金全額退還！
   (매주 미션 전부 완료 → 보증금 전액 환급!)

❌ 一週內有一天沒完成 → 該週扣除 1000元
   (한 주에 하루라도 못하면 → 그 주 1000원 차감)

[강조 포인트]
💡 首次參加 + 完成挑戰 = 完全免費學韓文！
   (첫 참여 + 완주 = 완전 무료로 한국어 배우기!)

   堅持21天，保證金全部退還給你
   (21일 완주하면, 보증금 전액 돌려받아요)

[완주율 언급]
📊 過去參加者完成率：85%以上！
   (기존 참가자 완주율: 85% 이상!)

   大部分的人都成功拿回保證金 💪
   (대부분 보증금 돌려받는 데 성공했어요 💪)
```

### 10. FAQ

**목적:** 망설임 해소

```
[헤딩]
常見問題
(자주 묻는 질문)

Q: 韓文程度要多好才能參加？
   (한국어 실력이 어느 정도여야 참가할 수 있나요?)
A: 基礎會話程度即可！會根據程度調整內容。
   (기초 회화 정도면 OK! 수준에 맞게 내용 조정해요.)

Q: 每天要花多少時間？
   (매일 얼마나 시간이 걸리나요?)
A: 約15-20分鐘。錄音+影片任務不會太長。
   (약 15-20분. 녹음+영상 미션은 오래 걸리지 않아요.)

Q: 如果某天真的沒辦法完成怎麼辦？
   (어떤 날 정말 못하면 어떡하죠?)
A: 一週內有一天沒完成，該週保證金會扣除1000元。
   但只要其他週完成，剩餘保證金還是會退還！
   (한 주에 하루 못하면 그 주 1000원 차감.
   다른 주 완료하면 나머지 보증금은 환급!)

Q: 通話會很緊張怎麼辦？
   (통화가 긴장되면 어떡하죠?)
A: 只有5分鐘，老師會引導對話，不用擔心！
   這也是練習的一部分 😊
   (5분뿐이고, 선생님이 대화 이끌어줘요. 걱정 마세요!
   이것도 연습의 일부예요 😊)

Q: 保證金怎麼退還？
   (보증금은 어떻게 환급되나요?)
A: 挑戰結束後，完成的週數對應金額會退還到你的帳戶。
   (챌린지 끝나면, 완료한 주차만큼 계좌로 환급돼요.)

Q: 首次參加的優惠之後還有嗎？
   (첫 참여 혜택은 나중에도 있나요?)
A: 參加費免除只限首次參加！第二次開始需要繳交參加費3000元。
   (참가비 면제는 첫 참여만! 두 번째부터는 참가비 3000원 있어요.)
```

## 디자인 가이드

### 톤앤매너
- 따뜻하고 친근한 느낌
- 모깨비 캐릭터 활용
- 파스텔 + 포인트 컬러

### 컬러
- 메인: 기존 브랜드 컬러 유지
- 강조: CTA 버튼, 가격, 완주율, 최초 참여 혜택

### 폰트
- 중국어: 가독성 좋은 산세리프
- 한국어: 포인트로 사용

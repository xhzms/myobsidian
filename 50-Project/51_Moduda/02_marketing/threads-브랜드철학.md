# Threads 콘텐츠: 브랜드 철학

> 상위 문서: [[threads-strategy]]
> 이 문서는 브랜드 철학 콘텐츠 제작 가이드입니다.

---

## 콘텐츠 목적

| 목적 | 설명 |
|------|------|
| **Why 전달** | 왜 이 서비스를 하는가 |
| **감정적 연결** | 브랜드에 대한 신뢰와 애정 |
| **차별화** | 단순 학습 서비스가 아님을 인식 |

---

## 핵심 원칙

### 문제-솔루션 vs 브랜드 철학

| 구분 | 문제-솔루션 | 브랜드 철학 |
|------|-------------|-------------|
| 질문 | **What** 해결하는가 | **Why** 하는가 |
| 초점 | 학습자의 문제 | 우리의 가치관/신념 |
| 톤 | 공감 + 해결책 | 스토리텔링 + 진정성 |

### 빈도 주의

- ❌ 자주 쓰면 피로감 → 특별함 상실
- ✅ 특별한 시점에만 (월 1~2회)
- ✅ 진정성 있을 때만

---

## 모두다코리안의 Why

### 핵심 철학 (인터뷰 기반)

> "외국어를 한다는 건 친구가 된다는 것"

| 키워드 | 의미 |
|--------|------|
| **관계** | 언어는 사람과 연결되는 도구 |
| **문화** | 문법이 아니라 문화를 알게 됨 |
| **따뜻함** | 외국어는 따뜻하다 |
| **자신감** | 못해도 괜찮아, 소통이 목적 |
| **재미** | 못한다고 선포하면 너무 재밌다 |

### 선생님 스토리

- 4개국어 학습 경험 (영어, 중국어, 스페인어, 한국어)
- 영어: 압박감 때문에 제일 못함
- 중국어/스페인어: 틀려도 괜찮은 환경 → 오히려 잘함
- 깨달음: **방식과 관점이 달랐다**

### 브랜드 미션

> "한국을 세계로 유통하는 브랜드"
> "사랑을 남기고 싶다"

---

## 다루는 주제

### 1. Why 모두다코리안
- 왜 이 서비스를 시작했는가
- 무엇을 믿는가
- 어떤 변화를 원하는가

### 2. 언어에 대한 철학
- 언어는 관계다
- 문법보다 소통
- 틀려도 괜찮아

### 3. 선생님 스토리
- 4개국어 학습 경험
- 대만과의 인연
- 학생들과의 에피소드

### 4. 학습자와의 관계
- 왜 말하기에 집착하는가
- 첫 번째 대화 상대가 되고 싶다
- 완주한 학생들의 변화

---

## 콘텐츠 구조 템플릿

```
[질문/화두 - Why에 대한 의문 제기]

[스토리/경험 - 개인적 이야기]

[깨달음 - 핵심 메시지]

[우리의 믿음 - 브랜드 철학]

[따뜻한 마무리 - 모깨비 톤]
```

---

## 실제 예시

### 예시 1: 왜 말하기에 집착하는가
```
"왜 우리는 '말하기'에 집착할까?"

사실 문법 가르치는 게 더 쉬워요.
교재 펴고 설명하면 되니까.

근데 우리는 매일 대화해요.
발음 하나하나 고쳐주고,
"이건 한국인은 이렇게 말해요" 알려주고.

왜?

한국어는 '관계'니까요.
누군가와 대화하고 싶어서 배우는 거잖아요.

우리는 그 첫 번째 대화 상대가 되고 싶어요 👻💙
```

### 예시 2: 외국어는 따뜻하다
```
"외국어는 따뜻하다"

이상한 말처럼 들리죠? ㅋㅋ

근데 진짜예요.

여행 가서 현지어로 인사하면
사람들 표정이 달라져요.

"어? 한국어 할 줄 알아요?"
그 순간 거리가 확 좁혀지는 느낌.

외국어를 한다는 건
친구가 된다는 거예요.

문법 완벽하게 하려고 배우는 거 아니잖아요.
누군가랑 대화하고 싶어서 배우는 거죠.

그래서 외국어는 따뜻해요 🌏💙
```

### 예시 3: 못해도 괜찮아
```
"못해도 괜찮아요"

영어, 중국어, 스페인어, 한국어.
4개 국어를 배워봤어요.

근데 제일 오래 한 영어가 제일 못해요.
왜냐면 "제대로 해야 해" 압박이 컸거든요.

중국어랑 스페인어는요?
틀려도 아무도 뭐라 안 해요.
그래서 오히려 편하게 말하게 됐어요.

깨달은 게 있어요.

👉 발음보다 중요한 건 소통이에요.
👉 못하는 거 당연한 거예요.
👉 학자 될 거 아니잖아요.

못한다고 선포하고 시작하면
언어가 너~~~무 재밌어져요 😆

모두다코리안이 그런 곳이 되고 싶어요.
틀려도 괜찮은, 편하게 말할 수 있는 곳 👻💙
```

### 예시 4: 언어는 문화다
```
"문법을 배우는 게 아니에요"

한국어를 배우면
문법만 느는 게 아니에요.

한국 문화를 알게 돼요.
한국 사람을 이해하게 돼요.
한국 콘텐츠가 2배로 재밌어져요.

"고구마"가 왜 답답하단 뜻인지
"치맥"이 왜 특별한 조합인지
"눈치"가 왜 번역이 안 되는지

이런 걸 알게 되는 거예요.

언어를 배운다는 건
그 나라의 문화를 접할 수 있는
기회가 커지는 거예요 🌏✨
```

## 중국어 버전 예시

```
"為什麼我們這麼執著於「說」？"

其實教文法比較輕鬆
翻開課本講解就好了

但我們每天都在對話
一個一個糾正發音
告訴你「韓國人其實是這樣說的」

為什麼？

因為韓文是「關係」
你學韓文，是因為想跟人對話吧？

我們想成為你第一個對話的對象 👻💙
```

## 적절한 타이밍

| 시점 | 예시 |
|------|------|
| 챌린지 완주 후 | 학생들 변화 이야기 |
| 연말/새해 | 한 해 돌아보기, 감사 |
| 브랜드 기념일 | 시작 스토리 |
| 특별한 에피소드 후 | 학생과의 감동 순간 |

---

## 주의사항

### ✅ Do
- 진정성 있는 스토리
- 개인적 경험 공유
- 따뜻한 톤
- 모깨비 캐릭터 유지

### ❌ Don't
- 자주 쓰기 (월 1~2회 권장)
- 서비스 홍보로 변질
- 뻔한 명언/인용
- 설교조

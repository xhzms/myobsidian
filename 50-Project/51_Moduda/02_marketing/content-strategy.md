## ⚠️ 콘텐츠 출력 규칙 (필수)

> **중요: 아래 규칙은 모든 콘텐츠 제작 시 반드시 준수!**

### 1. 이중 언어 출력 🌏🇰🇷

**타겟 언어로 콘텐츠 작성 시, 한국어 번역을 함께 제공합니다.**

```

[타겟 언어 콘텐츠]

```

**이유:** 담당자가 콘텐츠 내용을 이해하고 검토할 수 있도록

## 우리는 누구인가

**모두다코리안**은 대만 시장을 타겟으로 한 한국어 학습 서비스입니다.

| 항목     | 내용                                   |
| ------ | ------------------------------------ |
| 핵심 상품  | 한국어 매일 말하기 챌린지 → 1:1 과외              |
| 슬로건    | "한국인은 그렇게 안 말해요. 원어민과 말하기 학습 모두다코리안" |
| 브랜드 미션 | 한국을 세계로 유통하는 브랜드                     |

---

## 메인 타겟

### Primary: 중급 이상 학습자
- **특징**: 기본기는 갖췄지만 실전 대화 기회 부족
- **Pain Point**: 
  - "한국어 공부는 혼자인데, 같이 얘기할 사람이 없어요"
  - "한국 좋아하는 사람들이랑 교류하고 싶어요"

### 진짜 원하는 것 (내면의 핵심)
| 표면        | 내면                    |
| --------- | --------------------- |
| 말하기 실력 향상 | 원어민과 대화하는 기쁨          |
| 발음 교정     | 나와 취미가 맞는 사람과 교류하는 재미 |
| 학습 시스템    | 혼자가 아닌 함께하는 경험        |

### Secondary: 진지한 학습자
- 한국 대학원 진학 희망자
- 어학당 수강생/수료생

### 타겟 NOT
- 완전 초보자
- 가벼운 취미 수준

---

## 고객 변화 여정 (Before → After)

| Before           | After             |
| ---------------- | ----------------- |
| 혼자 공부, 말할 사람 없음  | 원어민과 대화하는 기쁨      |
| 한국 좋아하는데 나눌 곳 없음 | 취미 맞는 사람들과 교류     |
| 금방 포기            | 커뮤니티와 함께 완주       |
| 발음/표현 피드백 부재     | 실시간 교정 + 자연스러운 표현 |

---

## 서비스 특장점

1. **원어민과 진짜 대화** - 한국인과 소통하는 경험
2. **함께하는 커뮤니티** - 한국 좋아하는 사람들과 교류
3. **매일 말하기 습관** - 챌린지 시스템으로 꾸준함 유지
4. **실시간 피드백** - 발음 교정, 자연스러운 표현 코칭
5. **1:1 과외 연결** - 관계를 이어가는 심화 학습

## 핵심 메시지 프레임

### 가치 제안
> "교과서에서 배울 수 없는, 한국인이 진짜 쓰는 한국어"
> "혼자가 아닌, 함께 한국어로 대화하는 경험"

### 차별화 포인트
- 신조어/유행어 = 살아있는 한국어
- 원어민 = 진짜 발음과 뉘앙스 + **대화하는 기쁨**
- 커뮤니티 = **취미가 맞는 사람들과 연결**

### 행동 유도 메시지
- "한 달 후, 한국어로 대화하는 자신을 만나세요"
- "혼자 공부 말고, 함께 말해요"

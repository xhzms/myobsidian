## 콘텐츠 구조 템플릿

### 타입 A: 음식 추천형
```
[음식 소개 - 이미지 권장]

[음식 설명 / 먹는 방법]

[한국어 표현 연결 (선택)]

[질문 - 경험/추천 요청]
```

**예시:**
```
저메추 !

晚餐菜單推薦！
最近韓國正值泡菜季，配上現做的泡菜一起吃韓式燉豬肉（보쌈），
真的就是最頂級的一餐啊～ 😋🥬🐷
```

### 타입 B: 문화 설명형
```
[문화 현상 소개]

[배경/이유 설명]

[모깨비 경험 or TMI]

[질문으로 마무리]
```

**예시:**
```
🔥 불멍（bul-meong）

불멍 是 [由 불（火）＋ 멍때리다（放空、發呆）] 組成的韓國新造詞。

👉 靜靜看著火焰
👉 什麼都不想，讓腦袋休息一下

🌿 為什麼大家喜歡 불멍？
• 壓力大的時候
• 腦袋很亂的時候
• 想療癒、放鬆的時候

🔥 有人想跟我一起挑戰『這個冬天 불멍 힐링』嗎？ㅎㅎ
我昨天也才剛去 불멍 回來喔～🔥
```

### 타입 C: 절기/행사형
```
[오늘 날짜 / 행사 소개]

[한국 문화 설명]

[모깨비 연결 (캐릭터 활용)]

[한국어 표현 + 마무리]
```

**예시:**
```
啊！今天在韓國是『冬至』這個節氣耶！
你知道嗎？冬至其實是鬼怪最討厭的一天🥲
因為一年之中，今天的夜晚最長……

以前的人為了『趕走鬼怪』，會吃紅豆粥（팥죽）祈求避邪，
所以可能是因為這樣吧……我有點怕紅豆粥😱

可是……呼呼包（호빵）跟鯛魚燒（붕어빵）到底為什麼那麼好吃啦……？ㅋㅋㅋㅋㅋ
總之我今天先躲起來一下。
```

## 실제 예시 (전문)

### 붕어빵 + 붕세권
```
冷風吹得涼颼颼的季節，冬天！❄️
我們家附近新開了一家超特別的鯛魚燒店！

不只紅豆鯛魚燒、卡士達口味，
竟然還有披薩口味、起司口味、蘋果果醬口味，
甚至連像 Hotteok（韓式糖餅）一樣的口味都有！

口味多到爆炸，真的超有趣～🤤🔥
感覺很好吃吧？是不是很羨慕～嘿嘿嘿 😆

在韓文裡，像這樣家裡附近就有鯛魚燒天堂，
我們會叫這種地方叫做 **「붕세권 bung-se-gwon」**～
（붕 = 鯛魚燒 / 세권 = 生活圈）

另外，韓文裡也很常說 「역세권 yeok-se-gwon」，
意思就是「住在地鐵站生活圈裡」的地方～ 🚉✨
```

### 떡국 + 새해 문화
```
莫可比其實也很會做料理喔～
只要有萬能的鬼怪棒就行啦嘿嘿 👻✨

大家今天有吃年糕湯（떡국）嗎？
在韓國有一句話是：
新年（새해）的第一天一定要吃年糕湯（떡국），這樣才會長一歲～

那麼～莫可比活到現在，到底喝過幾碗年糕湯（떡국）了呢？ㅎㅎ
因為每喝一碗，就代表又長一歲喔！

雖然過年（春節）也會吃，
但 1 月 1 日的早上，
大家通常都是用年糕湯（떡국）來迎接新年（새해）ㅎㅎ

怎麼樣～看起來是不是很好吃呀？ 🤤🍲
```

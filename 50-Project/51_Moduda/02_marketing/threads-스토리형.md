# Threads 콘텐츠: 스토리텔링 형식

> 상위 문서: [[threads-strategy]]
> 이 문서는 스토리텔링 형식의 콘텐츠 제작 가이드입니다.

---

## 개요

스토리텔링은 **표현 형식**입니다. 모든 콘텐츠 유형에 적용할 수 있어요.

| 기본 유형 | 직접적 버전 | 스토리 버전 |
|-----------|-------------|-------------|
| 문제-솔루션 | 문제 → 원인 → 해결책 | 문제를 겪는 인물의 여정 |
| 교육/정보성 | 정보 → 설명 → 팁 | "이런 상황에서 이렇게 했더니" |
| 브랜드 철학 | 철학 → 이유 → 마무리 | 창업자/고객 실제 에피소드 |

---

## 콘텐츠 목적

| 목적 | 설명 |
|------|------|
| **공감 극대화** | 가상 인물을 통해 "나의 이야기"처럼 느끼게 |
| **자연스러운 전달** | 직접적 조언보다 거부감 ↓ |
| **기억에 남음** | 스토리는 정보보다 오래 기억됨 |

---

## 언제 사용하나?

### 트리거 키워드
- "스토리로 만들어줘"
- "사례로 풀어줘"
- "공감되는 이야기로"
- "에피소드 형식으로"

### 효과적인 상황
- 페인포인트가 감정적일 때
- 직접적 조언이 설교처럼 들릴 수 있을 때
- 독자가 "나도 그래" 느낌을 받았으면 할 때

---

## 콘텐츠 구조 템플릿

```
[도입] 인물 소개 + 상황 설정

[전개] 문제 직면 + 시도와 실패

[전환] 깨달음/발견

[결말] 변화/해결

[마무리] 독자에게 질문/공감 유도
```

---

## 구조별 작성 가이드

### 1. 도입: 인물 + 상황
- 타겟이 자신을 투영할 수 있는 인물
- 구체적인 상황 설정 (이름, 직업, 나이 등)
- 1~2문장으로 짧게

```
예시:
"小美(28)學韓文已經3年了。"
"大學生阿明每天都在背單字。"
```

### 2. 전개: 문제 + 시도
- 타겟이 겪는 문제 묘사
- 흔히 하는 실패/시도
- 감정 표현 포함

```
예시:
"奇怪的是，明明很努力，卻看不到進步。"
"試過這個方法，也試過那個。但還是一樣。"
```

### 3. 전환: 깨달음
- 문제의 원인 발견
- "아!" 하는 순간
- 핵심 인사이트

```
예시:
"有一天，她突然意識到了。"
"原來問題不是努力不夠，而是方法不對。"
```

### 4. 결말: 변화
- 깨달음 이후의 행동 변화
- 달라진 결과
- 희망적 톤

```
예시:
"從那之後，她開始每天練習說。"
"三個月後，終於體驗到了改變。"
```

### 5. 마무리: 독자 연결
- 독자에게 질문
- 공감 유도
- 모깨비 톤

```
예시:
"김서방들也有像小美一樣的經驗嗎？"
"你是不是也覺得「明明知道，卻做不到」？🤔"
```

---

## 인물 설정 가이드

### 이름 선택
| 타겟 | 이름 예시 |
|------|-----------|
| 대만 | 小美, 阿明, 小華, 小玲 |
| 한국 (참고용) | 민지, 수진, 준호, 지훈 |
| 중립 | "한 친구", "A씨", "有個人" |

### 구체성 레벨

| 레벨 | 예시 | 용도 |
|------|------|------|
| 낮음 | "有個朋友" | 가벼운 에피소드 |
| 중간 | "小美(28)" | 일반적 스토리 |
| 높음 | "學韓文3年的上班族小美(28)" | 타겟 특정 |

### 타겟 투영 원칙
- 인물 = 타겟 페르소나
- 나이, 직업, 상황이 타겟과 유사해야 공감
- 모두다코리안 타겟: 대만인 한국어 학습자 (20~30대)

## 실제 예시

### 직접적 버전 vs 스토리 버전

**직접적 버전:**
```
"學韓文3年，為什麼還是說不出口？"

文法書看了10本
單字背了5000個

但真的要說的時候… 🤐

原因很簡單
👉 「從來沒有真正說過」

真正的韓文，要說出口才會進步 🗣️👻
```

**스토리 버전:**
```
小美的故事

小美(28)學韓文已經3年了。
文法書看了10本，單字背了5000個。

「這樣應該夠了吧？」

但真的站在韓國人面前，嘴巴卻張不開。
腦袋一片空白 🤐

「為什麼？明明都知道的啊…」

有一天她突然意識到了。
只看游泳的書，不可能學會游泳。

從那之後，她開始每天練習說10分鐘。
三個月後，終於第一次成功跟韓國人對話了。

김서방들也有像小美一樣
「明明知道，卻做不到」的經驗嗎？🤔👻
```

### 예시 2: 발음 문제 스토리

```
阿明的煩惱

阿明很認真地練習韓文發音。
每天對著鏡子練習，覺得自己說得很標準。

有一天他終於鼓起勇氣跟韓國人說話。

「네? 다시요?」

對方聽不懂 😢

阿明很沮喪。「明明練習這麼久了…」

後來他才知道，
問題是「沒有人告訴他哪裡錯了」。

發音這種東西，要有人聽、有人糾正，才會進步。
對著鏡子練習，永遠不知道自己錯在哪。

김서방들有沒有過類似的經驗？
自以為對了，結果其實一直錯著？😅👻
```

### 예시 3: 자신감 문제 스토리

```
小玲的領悟

小玲學過4種外語。
英文、日文、韓文、西班牙文。

奇怪的是，學最久的英文反而最差。
為什麼？

因為從小被教育「英文要說得標準」
一說錯就被糾正，久了就不敢開口。

但韓文和西班牙文不一樣。
沒人期待她說得完美。
錯了也沒關係，反正就是在學嘛。

所以她反而敢說、敢錯、敢嘗試。

小玲終於明白了：
👉 不是能力問題，是心態問題。

「說不好是正常的」
接受這點之後，語言學習變得超～有趣 😆

김서방들是不是也被「要說得完美」綁住了？👻💙
```

## 중국어 버전 작성 팁

### 문장 톤
- 짧은 문장 유지
- 구어체 느낌
- 이모지로 감정 표현

### 김서방 활용
- 마무리에서 "김서방들" 호칭 사용
- 모깨비 캐릭터 연결

---

## 주의사항

### ✅ Do
- 타겟과 유사한 인물 설정
- 감정 표현 포함
- 희망적 결말
- 마무리에 독자 연결

### ❌ Don't
- 인물이 너무 완벽함
- 결말에서 서비스 홍보
- 너무 긴 스토리 (500자 제한!)
- 억지 교훈

---

## 체크리스트

- [ ] 인물이 타겟과 유사한가?
- [ ] 문제가 타겟의 실제 페인포인트인가?
- [ ] 전환점(깨달음)이 자연스러운가?
- [ ] 마무리에 독자 연결이 있는가?
- [ ] 모깨비/김서방 톤이 유지되었는가?
- [ ] 500자 이내인가?

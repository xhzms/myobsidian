# Threads 콘텐츠: 문제점-솔루션

> 상위 문서: [[threads-strategy]]
> 이 문서는 문제점-솔루션 콘텐츠 제작 가이드입니다.

---

## 콘텐츠 목적

| 목적 | 설명 |
|------|------|
| **문제 인식** | 타겟이 겪는 문제를 명확히 짚어줌 |
| **브랜드 포지셔닝** | 우리가 해결책임을 자연스럽게 인식 |
| **신뢰 구축** | "이 사람들은 내 문제를 이해하네" |

---

## 핵심 원칙

### 표면 vs 내면

| 레이어 | 다루는 것 | 언급 수준 |
|--------|-----------|-----------|
| **표면** | 말하기 연습, 실생활 표현 체득 | 직접 언급 ✅ |
| **내면** | 원어민과 대화하는 기쁨, 함께하는 경험 | 암시만 (직접 언급 ❌) |

### 판매 vs 가치

- ❌ 챌린지/서비스 직접 홍보 금지
- ✅ 문제 인식 + 방향 제시까지만
- ✅ "우리 서비스 써라"가 아니라 "이렇게 하면 된다"

---

## 콘텐츠 구조 템플릿

```
[후킹 - 문제 제기]

[공감 - "나도 그래" 느낌]

[원인 분석 - 왜 이런 문제가 생기는가]

[솔루션 방향 - 어떻게 해결할 수 있는가]

[모깨비 마무리 - 캐릭터 톤으로 정리]
```

---

## 구조별 작성 가이드

### 1. 후킹 (첫 줄)
```
"한국어 공부 3년, 근데 왜 말이 안 나올까?"
"TOPIK 6급인데, 한국인이 못 알아들어요"
"드라마 자막은 읽는데, 왜 웃는지 모르겠어요"
```
- 타겟의 좌절감을 정확히 짚기
- 질문형 or 상황 제시
- 짧고 강렬하게

### 2. 공감
```
문법책 10권 봤어요.
단어 5000개 외웠어요.
TOPIK도 땄어요.

근데 막상 한국인 앞에서… 🤐
```
- "나도 그래" 느낌
- 구체적인 상황 나열
- 이모지로 감정 표현

### 3. 원인 분석
```
이유는 간단해요.
👉 "말해본 적이 없으니까"

수영 책만 읽고 수영 잘하는 사람 없어요.
한국어도 마찬가지예요.
```
- 원인을 명확하게
- 비유 활용 (수영, 운전, 악기 등)
- 복잡하게 설명 ❌, 한 줄로 핵심

### 4. 솔루션 방향
```
진짜 한국어는 말해봐야 늘어요 🗣️
한국인이 쓰는 표현을 직접 써봐야 해요.
```
- 구체적 해결책 제시
- 단, 서비스 홍보는 ❌
- "이렇게 하면 된다" 방향만

### 5. 모깨비 마무리
```
한국인은 그렇게 안 말해요.
진짜 한국어, 말해볼 준비 됐어? 👻🔥
```
- 슬로건 자연 삽입 (선택)
- 캐릭터 톤 유지
- 질문형으로 마무리 (선택)

## 다루는 문제 카테고리

### A. 말하기/발음
| 문제 | 원인 | 솔루션 방향 |
|------|------|-------------|
| 공부했는데 말이 안 나와요 | 말해본 적이 없음 | 실제로 말해보기 |
| 발음이 이상하대요 | 피드백 없이 혼자 연습 | 원어민 교정 |
| 한국인이 못 알아들어요 | 억양/발음 문제 | 실시간 피드백 |

### B. 듣기/이해
| 문제 | 원인 | 솔루션 방향 |
|------|------|-------------|
| 자막 없으면 뉘앙스를 놓쳐요 | 표현의 감정/맥락 모름 | 실생활 표현 + 뉘앙스 학습 |
| 대사는 들리는데 왜 웃는지 모르겠어요 | 문화적 맥락/유행어 부족 | 살아있는 한국어 학습 |
| 신조어/줄임말 모르겠어요 | 교과서에 없는 표현 | 요즘 한국어 학습 |

### C. 자신감/심리적 장벽
| 문제 | 원인 | 솔루션 방향 |
|------|------|-------------|
| 틀릴까봐 말을 못하겠어요 | 지적하는 문화, 완벽해야 한다는 압박 | "못하는 거 당연해" 마인드 |
| 외국인 앞에서 머리가 하얘게 돼요 | 실전 경험 부족 | 부담 없이 말할 수 있는 환경 |
| 완벽하게 해야 할 것 같아요 | 학교 교육 방식 | 커뮤니케이션이 목적 |

### D. 학습 방법
| 문제 | 원인 | 솔루션 방향 |
|------|------|-------------|
| 뭘 더 해야 할지 모르겠어요 | 실전 경험 부족 | 말하기 중심 학습 |
| 혼자 공부하니 금방 포기 | 동기부여 부족 | 함께하는 학습 (암시) |
| 실력이 안 느는 것 같아요 | 아웃풋 부족 | 매일 말하기 습관 |

## 실제 예시

### 예시 1: 말해본 적 없음
```
"한국어 공부 3년, 근데 왜 말이 안 나올까?"

문법책 10권 봤어요.
단어 5000개 외웠어요.
TOPIK도 땄어요.

근데 막상 한국인 앞에서… 🤐

이유는 간단해요.
👉 "말해본 적이 없으니까"

수영 책만 읽고 수영 잘하는 사람 없어요.
한국어도 마찬가지예요.

한국인은 그렇게 안 말해요.
진짜 한국어는 말해봐야 늘어요 🗣️👻
```

### 예시 2: 드라마 뉘앙스
```
"드라마 자막은 읽는데, 왜 웃는지 모르겠어요"

대사는 이해했어요.
근데 다들 웃는데 나만 모르겠고… 😅

왜냐면요,
👉 단어의 '뜻'은 알아도 '느낌'을 모르니까

"고구마"가 지과(sweet potato)인 건 알아요.
근데 왜 답답한 상황에 "고구마"라고 하는지는 몰랐죠.

자막으로는 절대 전달 안 되는 것들이 있어요.
표현의 느낌, 감정, 맥락 같은 거요.

이걸 알면 드라마가 2배로 재밌어져요 🎬✨
```

### 예시 3: 발음 문제
```
"열심히 말했는데 한국인이 못 알아들어요"

분명 맞게 말한 것 같은데
"네? 다시요?" 들을 때 그 기분… 😢

문제는 대부분
👉 혼자 연습해서 틀린 줄 몰랐던 것

발음은 누군가 들어주고
바로바로 고쳐줘야 제대로 잡혀요.

거울 보고 혼자 연습? 한계 있어요.
진짜 한국인한테 들려봐야 알 수 있어요 🗣️👻
```

### 예시 4: 자신감 문제
```
"문법은 아는데 왜 말이 안 나올까?"

영어 10년 공부했어요.
근데 제일 못해요.

중국어, 스페인어는 시간 덤 들였는데
오히려 간단한 회화가 더 수월해요.

차이가 뭐였냐면요,
👉 압박감

영어는 "제대로 해야 해" 압박이 커서
틀리면 지적받을까봐 말을 못했어요.

근데 생각해보면요,
못하는 거 당연한 거잖아요.
한국 사람이 중국어 못하는 거 당연하듯이.

문법 틀려도 소통만 되면 되는 거예요.
못한다고 선포하고 하면, 너~~~무 재밌어요 😆👻
```

## 중국어 버전 예시

```
"學韓文 3 年，為什麼還是說不出口？"

文法書看了 10 本
單字背了 5000 個
TOPIK 也考過了

但真的站在韓國人面前… 🤐

原因很簡單
👉 「因為從來沒有真正說過」

只看游泳的書，不可能學會游泳
韓文也是一樣的道理

한국인은 그렇게 안 말해요.
真正的韓文，要說出口才會進步 🗣️👻
```

## 주의사항

### ✅ Do
- 타겟의 좌절감에 공감
- 원인을 명확하고 간단하게
- 희망을 주는 솔루션 방향
- 모깨비 캐릭터 유지

### ❌ Don't
- 서비스/챌린지 직접 홍보
- 문제만 지적하고 끝 (해결책 없음)
- 너무 길게 설명
- 타겟 비하 ("그러니까 안 되는 거야")

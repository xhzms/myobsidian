## 주제 카테고리

### 1. 계절/날씨
```
我冷死啦～～莫可比要結冰了 ㅠㅠ🥶
聽說韓國這週四要下雪了欸！
台灣也會下雪嗎？❄️
```

### 2. 일상 다짐 (실패 공감)
```
不是啦～莫可比本來想說從新年開始要運動的……
結果……一轉眼都已經過 5 天了……！
誰可以直接把我「拎」去健身房、放在那裡就好嗎？
我真的……腳步怎麼樣都邁不出去……OTL 🥲💪
```

### 3. 연말/새해
```
轉眼間又到年末了呢 ✨
今年大家真的辛苦了。

謝謝你這一年陪莫可比一起玩、一起學韓語 👻
新的一年也請多多指教～
希望 2026 年能帶給你更多溫暖和好事 💫
莫可比先抱一下再走！👻🫂
```

### 4. 먹방/야식
```
明明要睡覺了…
結果在 Threads 看到草莓刨冰，腦袋直接清醒😂
冬天吃雪冰的草莓刨冰真的就是最強…
拜託大家去吃 #雪冰\_草莓刨冰 🍓✨
```

### 5. 질문형 (참여 유도)
```
哇，竟然已經 12 月了！
大家年底／跨年計畫怎麼安排的？
莫可比打算來個《小鬼當家》大重溫 😭
台灣年末年初都有哪些活動啊？好好奇！
```

## 콘텐츠 구조 템플릿

### 타입 A: 일상 토로형
```
[상황 공유]

[감정 표현 - 이모지 활용]

[공감 or 질문으로 마무리]
```

**예시:**
```
昨天是 1 月 1 日新年（新正）的國定假日，休息了一天，
結果今天星期五就要上班，感覺有點硬撐呢😵‍💫

大家今天上班／上課都還順利嗎？

昨天才立下的新年新希望，
大家沒有馬上就忘記吧？
作三天就放棄的人（작심삼일러）NO NO！ 🙅‍♂️

莫可比呢，是打算從下週開始運動～
所以現在還在「立志前階段」ㅋㅋㅋ

今天大家也一起加油吧！💪👻✨
```

### 타입 B: 계절/날씨형
```
[날씨 언급]

[한국 상황 공유]

[대만 상황 질문]
```

### 타입 C: 연말/기념일형
```
[인사/감사]

[모깨비 캐릭터로 따뜻한 마무리]

[한국어 표현 소개 (선택)]
```

## 공감 포인트 리스트

| 주제 | 공감 포인트 | 예시 표현 |
|------|-------------|-----------|
| 운동 | 다짐만 하고 안 함 | "從下週開始運動～" |

| 휴일 후 출근 | 힘듦 | "感覺有點硬撐" |

## 모깨비 캐릭터 활용

### 캐릭터 특성 반영
- **허당**: 운동 미루기, 야식 유혹에 약함
- **따뜻함**: 연말 인사, 응원 메시지
- **장난기**: 가벼운 셀프 디스

### 캐릭터 표현 예시
```
莫可比先抱一下再走！👻🫂
```

```
我也想成為瘦瘦的鬼怪啦👻
```

```
所以現在還在「立志前階段」ㅋㅋㅋ
```

# 메시지 프레임워크

## 핵심 메시지 (Core Message)
> "한국인은 그렇게 안 말해요. 원어민과 말하기 학습 모두다코리안"

## 브랜드 포지셔닝
- **카테고리**: 한국어 회화 학습 서비스
- **차별점**: 원어민이 직접 진행 + 매일 실전 대화
- **타겟 약속**: 한 달 후, 한국어로 대화하는 자신을 만나세요

---

## 브랜드 캐릭터: 모깨비 (莫可比) 👻

### 캐릭터 설정
- **정체**: 도깨비 (鬼怪)
- **상징**: 👻🔥 (불꽃 도깨비)
- **도구**: 도깨비 방망이 (뚝딱 마법), 감투 (隱身帽)
- **성격**: 친근함, 장난기, 따뜻함

### 말투 특징
- 1인칭: "莫可比", "모깨비"
- 반말 + 존댓말 자연스럽게 혼용
- "~喔", "~啦", "ㅎㅎ", "ㅋㅋ" 적극 사용
- 이모지 풍부하게 활용

### 캐릭터 활용 예시
```
莫可比其實也是뚜벅이啦～
但晚上會變身成火焰飛來飛去，這個算是不公開的小秘密ㅎㅎ👻🔥
```

## 콘텐츠 유형별 메시지 패턴

### 1. 신조어/유행어 교육 (메인 콘텐츠)

**구조:**
```
🔥 [단어] [발음]

[정의/의미 설명 - 중국어]

👉 핵심 뜻 1줄 요약

💬 예문 2~3개

✨ 모깨비 총정리 / TMI

[질문으로 마무리 - 댓글 유도]
```

**예시:**
```
🍠 고구마 goguma (地瓜)

吃 고구마 如果不配水，喉嚨會被卡住、覺得超悶對吧 😖
👉 很悶、很卡、讓人快爆炸的狀況

💬 例句
• 這情況真的很 고구마。
• 這一集是 고구마 劇情 ㅠㅠ

✨ 我現在的心情就像吃了 100 個 고구마。

고구마的反義詞要怎麼說呢？下一篇公布ㅎㅎ
```

### 2. 챌린지 모집/홍보

**구조:**
```
[후킹 - 문제 제기 or 혜택]

[챌린지 특징 3~5개]

[타겟 명시 - "이런 분에게 추천"]

[CTA - 링크/마감 안내]
```

**예시:**
```
12월「每天韓文口說挑戰」開始招募囉～！

很適合這樣的你：
1️⃣ 學韓文很久了，但開口說還是有點可惜的人
2️⃣ 想常常說韓文，卻沒有說話機會的人
3️⃣ 想學韓國人日常真的會用的表達方式的人

報名請參考個人簡介上的連結喔！
```

### 3. 일상/공감

**구조:**
```
[모깨비 일상 공유]

[공감 포인트]

[질문으로 마무리]
```

**예시:**
```
不是啦～莫可比本來想說從新年開始要運動的……
結果……一轉眼都已經過 5 天了……！
誰可以直接把我「拎」去健身房？🥲💪
```

## 후킹 공식 (Threads 최적화)

### 1. 충격 + 믿음 뒤집기
- "한국어 공부, 열심히 해도 안 느는 이유"
- "문법책 100권보다 원어민 10분이 나은 이유"

### 2. 궁금증 + 결과 암시
- "이 습관 하나로 한국어 실력이 달라졌습니다"
- "한 달 만에 한국인 친구가 생긴 비결"

### 3. 비밀 + 희소성
- "한국어 고수들만 아는 말하기 연습법"
- "95%는 아직 모르는 한국 신조어"

### 4. 질문형 마무리 (댓글 유도)
- "你們心目中「맛없없」的韓國美食是哪一個？"
- "大家最近的神曲推薦是什麼呢？"
- "快留言告訴莫可比吧～💙"

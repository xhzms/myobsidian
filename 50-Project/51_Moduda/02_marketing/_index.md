
## 사용 방법

### 콘텐츠 제작시
1. **[[content-strategy]]** - 전체 방향 확인
2. **[[threads-strategy]]** - 채널 특화 전략 확인
3. **유형별 문서** - 제작 가이드 참조

### 스킬즈 트리거 예시
```
"모깨비 스레드 콘텐츠 제작 - 신조어"
→ content-strategy → threads-strategy → threads-신조어교육
```

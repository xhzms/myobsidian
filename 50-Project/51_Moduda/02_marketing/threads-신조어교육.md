## 주제 선정 기준

### 좋은 주제
- 드라마/예능에서 자주 등장
- 일상에서 바로 쓸 수 있음
- 공감 포인트가 있음 (감정, 관계, 음식)
- 시리즈로 연결 가능 (반의어, 관련어)

### 피해야 할 주제
- 너무 오래된 유행어
- 설명이 복잡한 단어
- 부정적/논란 가능성 있는 표현


## 구조별 작성 가이드

### 1. 제목부 (첫 줄)
```
🍠 고구마 goguma (地瓜)
🔥 완내스 完內斯（wan-nae-seu）
💸 내돈내산（nae-don-nae-san）
```
- 이모지로 시선 끌기
- 한글 + 발음 + 중국어 병기
- 발음은 영어 로마자 또는 주음 활용

### 2. 정의/의미
```
吃 고구마 如果不配水，
喉嚨會被卡住、覺得超悶對吧 😖
這個說法就是從那種「悶到不行」的感覺來的～
```
- 비유나 상황으로 쉽게 설명
- 너무 길지 않게 (3~5문장)
- 이모지로 감정 표현

### 3. 핵심 뜻 요약
```
👉 很悶、很卡、讓人快爆炸的狀況或言行
```
- 한 줄로 정리
- 화살표(👉)로 강조

### 4. 예문
```
💬 例句
• 哇……這情況真的很 고구마。
• 這一集是 고구마 劇情，看得好痛苦 ㅠㅠ
• 主角的個性太 고구마 了，好讓人煩。

🔹 加強說法
• 我現在的心情就像吃了 100 個 고구마。
```
- 일상 상황에서 바로 쓸 수 있는 예문
- 드라마/SNS 상황 활용
- 강조 표현도 함께

### 5. 모깨비 총정리
```
✨ 모깨비式總結
고구마 = 超級悶、讓人快爆炸的感覺
下一篇要來講반義詞요～ㅎㅎ
```
- 캐릭터 말투 유지
- 다음 편 예고 (시리즈 연결)
- TMI나 개인 경험 추가 가능

**⚠️ 주의: "자주 쓴다" 표현 남용 금지**
- ❌ 모든 신조어에 "모깨비도 자주 써요" → 신뢰도 저하
- ✅ 실제 빈도에 맞게 표현 선택:
  - 자주 쓰는 표현: "모깨비도 진짜 자주 써요"
  - 가끔 쓰는 표현: "모깨비도 가끔 써요" / "써본 적 있어요"
  - 상황 특정: "~할 때 써요" / "~보면 생각나요"
  - 공감형: "이 표현 들으면 공감돼요" / "이런 상황 있죠?"

### 6. 질문형 마무리
```
고구마的反義詞要怎麼說呢？答案是?!
下一篇我會用『요즘한국어』的形式發文喔ㅎㅎ
我剛剛是不是有點像 고구마？ㅎㅎㅎㅎ
```
- 댓글 유도
- 다음 편 기대감

## 카테고리별 예시 단어

### 감정/상태
| 단어 | 뜻 | 시리즈 연결 |
|------|-----|-------------|
| 고구마 | 답답함 | → 사이다 |
| 사이다 | 시원함 | ← 고구마 |
| 멘붕 | 정신 붕괴 | → 현타 |
| 심쿵 | 심장 쿵 | 단독 |
| 킹받네 | 빡침 | 단독 |

### 음식/맛
| 단어 | 뜻 | 시리즈 연결 |
|------|-----|-------------|
| 맛없없 | 맛없을 수 없다 | 단독 |
| 존맛탱 | 존나 맛있다 | 단독 |
| 쩝쩝박사 | 맛 조합 고수 | → 맛잘알 |
| 맛잘알 | 맛 잘 아는 사람 | ← 쩝쩝박사 |
| 먹킷리스트 | 먹을 것 버킷리스트 | 단독 |

### 관계/연애
| 단어 | 뜻 | 시리즈 연결 |
|------|-----|-------------|
| 삼귀다 | 썸+사귀다 | 단독 |
| 어장관리 | 여러 명 관리 | 단독 |

### 생활/문화
| 단어 | 뜻 | 시리즈 연결 |
|------|-----|-------------|
| 혼밥 | 혼자 밥 | → 혼코노 |
| 혼코노 | 혼자 코인노래방 | ← 혼밥 |
| 갓생 | 갓+인생 | 단독 |
| 국룰 | 국민 룰 | 단독 |
| 호캉스 | 호텔+바캉스 | → 찜캉스 |

### 취향 표현
| 단어 | 뜻 | 시리즈 연결 |
|------|-----|-------------|
| 갠취 | 개인 취향 | → 취저 |
| 취저 | 취향 저격 | ← 갠취, → 완내스 |
| 완내스 | 완전 내 스타일 | ← 취저 |

## 실제 예시 (전문)

### 예시 1: 고구마
```
🍠 고구마 goguma (地瓜)

吃 고구마 goguma (地瓜) 如果不配水，
喉嚨會被卡住、覺得超悶對吧 😖
這個說法就是從那種「悶到不行」的感覺來的～

那種讓人很憋、很煩的感覺
＝ 就像在吃 고구마 goguma (地瓜) 一樣 🍠

所以現在韓國會用 고구마 goguma (地瓜) 來形容
很悶、很卡、讓人快爆炸的狀況或言行。

✨ 使用例句
• 哇……這情況真的很 고구마 goguma (地瓜)。
• 這一集是 고구마 goguma (地瓜) 劇情，看得好痛苦 ㅠㅠ
• 主角的個性太 고구마 goguma (地瓜) 了，好讓人煩。

🔹 加強說法
• 我現在的心情就像吃了 100 個 고구마 goguma (地瓜)。
• 這該不會是故意在 餵고구마 goguma (地瓜) 的劇情吧？

고구마(地瓜) 的反義詞要怎麼說呢？答案是?!
下一篇我會用『요즘한국어』的形式發文喔ㅎㅎ
我剛剛是不是有點像 고구마(地瓜)？ㅎㅎㅎㅎ
```

### 예시 2: 국룰
```
🧩국룰是？[gung-nul]

就是大家心照不宣的「不成文規則」。
白話說：不用講，但不照做會被翻白眼那種😎

🎯 使用方式
句尾加上 「……，국룰。」
就是迷因完成！語氣放輕鬆，像跟朋友點頭即可 ✌️

[梗味例句（可直接複製）]
🍗 炸雞點辣醬半份＋原味半份，국룰。
🍺 去漢江就是雞配啤酒，국룰。
🌧️ 下雨天煎餅＋瑪格利酒，국룰。
🎬 電影院先買爆米花，국룰。

👍一鍵模板
[場所/情境] = [固定組合]，국룰。
[事件] 後要 [認證/儀式]，국룰。

💬 可愛程度≥莫可比，必須收藏，國盧？📌
```

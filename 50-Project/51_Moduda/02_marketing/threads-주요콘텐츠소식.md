# Threads 콘텐츠: 주요 콘텐츠 소식

> 상위 문서: [[threads-strategy]]
> 이 문서는 주요 콘텐츠 소식 제작 가이드입니다.

---

## 콘텐츠 목적

| 목적 | 설명 |
|------|------|
| **공감** | "나도 봤어!" 연결감 |
| **트렌드 참여** | 화제에 함께하는 브랜드 |
| **한국 문화 연결** | 콘텐츠 → 한국어/문화 학습 연결 |

---

## 제작 방식: 티키타카 플로우

> 작업자와 대화하며 콘텐츠를 만들어가는 방식

```
[Claude] 질문
    ↓
[작업자] 답변
    ↓
[Claude] 후속 질문 (반복)
    ↓
[Claude] 콘텐츠 초안
    ↓
[작업자] 피드백
    ↓
[Claude] 수정
    ↓
[작업자] 확정
```

---

## 질문 플로우

### Step 1: 콘텐츠 발굴

**시작 질문 (택 1)**
```
"요즘 뭐 보고 계세요? (드라마/예능/영화/유튜브)"
"최근에 본 것 중에 재밌었던 거 있어요?"
"요즘 화제인 콘텐츠 뭐가 있어요?"
"SNS에서 많이 보이는 콘텐츠 있어요?"
```

**타이밍 트리거 질문**
```
"이번 주 넷플릭스 신작 중에 볼 만한 거 있어요?"
"요즘 음악방송에서 누가 핫해요?"
"최근 영화 개봉작 중에 본 거 있어요?"
```

---

### Step 2: 배경/맥락 파악

**전작/시리즈 질문**
```
"시즌1도 보셨어요?"
"전작이랑 비교하면 어때요?"
"이 감독/배우 다른 작품도 봤어요?"
```

**시청 맥락 질문**
```
"혼자 보세요 아니면 누구랑 같이 봐요?"
"어떤 계기로 보게 됐어요?"
"주변에서 추천받았어요?"
```

---

### Step 3: 반응/감상 추출

**감상 질문**
```
"어떤 점이 제일 재밌었어요?"
"인상 깊었던 장면이나 순간이 있어요?"
"보면서 어떤 감정이 들었어요?"
```

**구체화 질문**
```
"제일 좋아하는 캐릭터/출연자가 누구예요?"
"명장면을 하나 꼽는다면?"
"친구한테 추천한다면 뭐라고 말할 것 같아요?"
```

**감정 심화 질문**
```
"웃겼어요? 감동받았어요? 긴장됐어요?"
"보면서 배고파졌어요? ㅋㅋ (음식 관련)"
"다음 회 기다려져요?"
```

---

### Step 4: 디테일 파고들기

**에피소드/장면 질문**
```
"그 장면 좀 더 설명해주세요. 무슨 상황이었어요?"
"그때 누가 뭐라고 했어요?"
"왜 그게 웃겼어요/감동이었어요?"
```

**출연자/캐릭터 질문**
```
"그 사람 어떤 스타일이에요?"
"케미가 좋다고 했는데, 어떤 관계예요?"
"티격태격한다는 게 어떤 느낌이에요?"
```

---

### Step 5: 한국어/문화 연결

**표현 연결 질문**
```
"이 콘텐츠에서 자주 나오는 한국어 표현 있어요?"
"제목이나 대사 중에 설명해줄 만한 거 있어요?"
"관련된 신조어나 밈이 있어요?"
```

**문화 연결 질문**
```
"이 콘텐츠가 왜 한국에서 인기인 것 같아요?"
"대만 사람들이 모를 만한 한국 문화 포인트가 있어요?"
"한국에서만 통하는 유머나 맥락이 있어요?"
```

---

### Step 6: 콘텐츠 마무리

**마무리 질문**
```
"팔로워들한테 질문 던진다면 뭘 물어볼까요?"
"이 콘텐츠 한 줄 요약하면?"
"모깨비가 이거 봤다면 뭐라고 할 것 같아요?"
"댓글 많이 달릴 것 같은 질문이 뭘까요?"
```

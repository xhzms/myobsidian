# 티키타카 콘텐츠 제작 플로우

## ⚠️ 트렌드/공감 콘텐츠 전용

이 방식은 **트렌드/공감 콘텐츠에만 사용**합니다.
다른 콘텐츠 유형(교육, 문제-솔루션, 브랜드 철학)은 직접 제작 방식을 사용하세요.

### 왜 트렌드 콘텐츠에만?
- 트렌드 콘텐츠는 **작업자의 생생한 반응**이 핵심
- 화제 콘텐츠에 대한 실제 경험과 감상이 필요
- 다른 유형은 전문 지식/브랜드 스토리 기반이라 직접 제작이 효율적

---

## 질문 플로우

### Step 1: 주제 발굴
```
"요즘 [업계/타겟] 관련해서 화제인 거 있어요?"
"최근에 인상 깊었던 [관련 콘텐츠/이슈] 있어요?"
```

### Step 2: 배경/맥락 파악
```
"이게 왜 화제가 됐어요?"
"평소에도 관심 있던 주제예요?"
```

### Step 3: 반응/감상 추출
```
"어떤 점이 제일 인상적이었어요?"
"보면서/들으면서 어떤 생각이 들었어요?"
```

### Step 4: 디테일 파고들기
```
"그 부분 좀 더 설명해주세요."
"구체적으로 어떤 상황이었어요?"
```

### Step 5: 타겟 연결
```
"이게 우리 타겟한테 어떤 의미가 있을까요?"
"타겟이 모를 만한 인사이트가 있어요?"
```

### Step 6: 콘텐츠 마무리
```
"팔로워들한테 질문 던진다면 뭘 물어볼까요?"
"댓글 많이 달릴 것 같은 질문이 뭘까요?"
```

## 플로우 예시

```
[Claude] 요즘 화제인 콘텐츠 있어요?

[작업자] [콘텐츠명] 봤어요

[Claude] 오 그거 요즘 화제던데! 어떤 점이 인상적이었어요?

[작업자] [구체적 반응]

[Claude] 그 부분 좀 더 설명해주세요.

[작업자] [디테일 설명]

[Claude] 우리 타겟한테 연결할 포인트가 있을까요?

[작업자] [연결 아이디어]

[Claude] 팔로워들한테 뭘 물어볼까요?

[작업자] [질문 제안]

[Claude] 좋아요! 콘텐츠 초안 작성할게요!

---
[초안]
---

[작업자] 좋은데 [수정 포인트] 바꿔주세요

[Claude] 수정할게요! → 확정
```

## 타이밍 가이드

| 트렌드 유형 | 최적 타이밍 |
|-------------|-------------|
| 뉴스/발표 | 당일 ~ 다음날 |
| 바이럴 | 확산 시점 ~ 3일 |
| 시즌 이벤트 | 해당 시즌 |

**핵심:** 화제일 때 빠르게!

# Funnel Design Skill

IT 서비스 프로젝트의 마케팅 퍼널 설계와 콘텐츠 제작을 지원하는 스킬입니다.

## 스킬 선택

사용자 요청에 따라 적절한 가이드를 참조합니다:

| 요청 유형 | 가이드 | 트리거 키워드 |
|-----------|--------|---------------|
| 브랜드 자산 추출 | `00_brand-extraction/` | 웹사이트 분석, 브랜드 추출, 자산 수집, 경쟁사 분석 |
| 가이드 문서 작성 | `01_project-setup/` | 프로젝트 세팅, 퍼널 설계, 문서 작성, 폴더 구조 |
| SNS 콘텐츠 제작 | `02_content-creation/` | 콘텐츠 만들기, 포스팅, SNS, 티키타카 |
| **랜딩페이지 기획** | `02_content-creation/landing-page.md` | 랜딩페이지, LP, 전환 페이지 |

## 가이드

### 00_brand-extraction
웹사이트에서 브랜드 자산(이미지, 텍스트, 폰트, 색상)을 추출합니다.
- 크롤러로 페이지 탐색 + 자산 다운로드
- 이미지 용도별 분류 (hero, product, icon 등)
- 애셋 리스트 생성 (brand-assets.md)
- 브랜드 분석 리포트 생성 (brand-analysis.md)

→ 상세: [00_brand-extraction/GUIDE.md](00_brand-extraction/GUIDE.md)

### 01_project-setup
프로젝트 초기 세팅부터 마케팅 가이드 문서까지 작성합니다.
- 폴더 구조 생성
- 사업 골조 문서 (비즈니스 모델, 타겟, 로드맵)
- 퍼널 전략 문서
- 마케팅 가이드 (Cascading 구조)

→ 상세: [01_project-setup/GUIDE.md](01_project-setup/GUIDE.md)

### 02_content-creation
작성된 가이드를 기반으로 실제 콘텐츠를 제작합니다.
- 티키타카 방식 콘텐츠 제작
- 콘텐츠 유형별 제작 (교육, 트렌드, 문제솔루션, 브랜드철학)

⚠️ **프로젝트 문서 필요**: 콘텐츠 제작 시 프로젝트 폴더가 참조되지 않았다면, 먼저 프로젝트 경로를 물어보고 관련 문서(타겟, 콘텐츠 전략, 채널 가이드)를 로드해야 합니다.

→ 상세: [02_content-creation/GUIDE.md](02_content-creation/GUIDE.md)

## 작업 흐름

```
[Phase 0] 브랜드 분석 → 00_brand-extraction (선택)
[Phase 1] 프로젝트 세팅 → 01_project-setup
[Phase 2] 사업 골조 정의 → 01_project-setup
[Phase 3] 퍼널 설계 → 01_project-setup
[Phase 4] 마케팅 가이드 → 01_project-setup
[Phase 5] 콘텐츠 제작 → 02_content-creation (프로젝트 문서 로드 필수)
```

## 핵심 원칙

### 표면 vs 내면
고객의 표면적 니즈와 내면적 니즈를 구분합니다:
- **표면**: 직접 언급 ✅ (스킬 향상, 문제 해결)
- **내면**: 암시만 💭 (관계, 소속감, 재미)

### Cascading 문서 구조
```
Level 1: 전체 전략 (content-strategy.md)
    ↓
Level 2: 채널별 전략 (channel-strategy.md)
    ↓
Level 3: 콘텐츠 유형별 가이드 (content-type.md)
```

### 인지 단계 콘텐츠 믹스
| 유형 | 비율 | 역할 |
|------|------|------|
| 교육/정보성 | 40% | 가치 제공 |
| 트렌드/공감 | 30% | 관계 형성 |
| 문제-솔루션 | 20% | 포지셔닝 |
| 브랜드 철학 | 10% | 감정 연결 |
